🧠 MeetingIQ — Organizational Intelligence PlatformMeetingIQ is an AI-powered organizational intelligence platform that transforms raw meeting transcripts, documents, and audio recordings into structured, actionable insights. Built with Streamlit, Groq (LLaMA 3.3 & Whisper), and ChromaDB, it automatically extracts and tracks projects, action items, escalations, risks, and critical decisions.It features a specialized Duplicate Escalation Detection & Tracking System to flag recurring issues across different meetings and an automated email engine to route real-time operational alerts directly to managers.🚀 Key FeaturesMulti-Modal Ingestion: Paste raw text, upload documents (.txt, .pdf, .docx), or upload audio files directly (.mp3, .wav, .m4a, etc.) for automatic transcription via Whisper-Large-V3.AI Extraction Engine: Powered by LLaMA 3.3 (70B) to parse unstructured text into highly accurate, structured JSON schemas.Intelligent Duplicate Escalation Tracker: Automatically detects if an incoming escalation matches a pre-existing open issue using fuzzy text matching ($>75\%$ similarity). It appends tracking history directly to the parent record and dynamically bumps severity tiers if necessary.Natural Language Query (RAG): Talk directly to your meeting database. Uses ChromaDB vector embeddings (all-MiniLM-L6-v2) combined with structured SQL constraints to provide hallucination-free answers about tasks, blockers, and decisions.Operational Email Routing: Automatically dispatches structured Markdown/Text emails using Gmail SMTP whenever new meetings are processed or action items/escalations are marked resolved.🛠 Tech StackFrontend UI: Streamlit (Custom Dark Theme UI)LLM & Audio Transcription: Groq API (llama-3.3-70b-versatile & whisper-large-v3)Vector Database (RAG): ChromaDB & HuggingFace Embeddings (sentence-transformers/all-MiniLM-L6-v2)Structured Database: SQLite3 + PandasEmail Automation: Python smtplib & email.mime📂 Project ArchitecturePlaintext├── app.py                 # Core Streamlit application & backend pipelines

⚠️ Note on Email Notifications: GMAIL_PASS must be a Gmail App Password, not your standard login password. You can generate this under your Google Account Security settings if 2-Factor Authentication is active.4. Run the ApplicationBashstreamlit run app.py
The application will automatically initialize the local databases and launch in your default web browser at http://localhost:8501.📊 Database Schema BreakdownsMeetingIQ stores analytics across 7 interconnected tables within SQLite:meetings: Core tracking ledger for uploaded assets.projects: Ongoing corporate initiatives, descriptions, and statuses.action_items: Tasks, owners, priority flags, and deadlines.escalations: High-priority operational bottlenecks with resolution tracking.risks: Risk registry documenting cross-team dependencies and downstream impacts.decisions: Central log capturing explicit choices, rationales, and owners.stakeholders: Rolodex compiling team participation frequency across events.🔒 Security & Data PrivacyLocal Storage: Transcripts, embeddings, vector indices, and relational data are stored completely locally on your machine via SQLite and ChromaDB.API Security: Data payloads sent to Groq are strictly bound to processing workflows governed by your private API keys.
